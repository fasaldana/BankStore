# 📱 Product Manager App

Una aplicación de React Native para gestionar productos. Permite listar, crear, actualizar y eliminar productos de forma sencilla.

## 🚀 Tecnologías utilizadas

- **React Native** - Framework para aplicaciones móviles
- **Expo** - Plataforma para desarrollar y ejecutar aplicaciones React Native
- **Redux** - Manejo del estado global de la aplicación

## 📦 Instalación

Asegúrate de tener **Node.js** y **Expo CLI** instalados en tu sistema.

1. Clona este repositorio:

   ```sh
   git clone https://github.com/tu-usuario/product-manager-app.git
   ```

2. Entra en la carpeta del proyecto:

   ```sh
   cd product-manager-app
   ```

3. Instala las dependencias:

   ```sh
   npm install
   ```

4. Inicia la aplicación:

   ```sh
   expo start
   ```

## 🛠️ Funcionalidades

✅ Ver lista de productos  
✅ Buscar productos por nombre  
✅ Agregar nuevos productos  
✅ Editar productos existentes  
✅ Eliminar productos  
✅ Gestión de estado con Redux  

## 🧪 consideraciones

En el archivo api/product.ts se debe configurar la url del host para hacer pruebas locales en dispositivos móviles.

reemplazar
```sh
http://localhost:3002
```
por el ip del dispositivo.

Ejemplo:

```sh
http://192.168.x.x:3002
```
